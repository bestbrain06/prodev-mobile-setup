# Setting Up and Testing My Mobile Development Environment

## Objective

To set up a React Native mobile development environment using the Expo Framework and test it using Expo Go on a physical device.

## Steps I Followed

1. Installed Node.js (LTS version) from [https://nodejs.org](https://nodejs.org).
2. Installed Visual Studio Code as my IDE.
3. Installed the Expo Go app:
   - Android: from Google Play Store
   - iOS: from Apple App Store
4. Created a new Expo project using:
   ```bash
   npx create-expo-app my-first-app
   ```
